# BumpBudget

A web and mobile app helping expecting and new parents manage finances around pregnancy and early parenthood.

## Built With
- Django
- MySQL
- JavaScript
- Chart.js
- Pandas
- scikit-learn

## Features
- User accounts and secure authentication
- Budget management and expense tracking
- Maternity leave income forecasting using scikit-learn and Pandas
- Milestone tracking
- Cost comparisons
- Community features
- Visual financial dashboards using Chart.js

## Status
In progress — Final Year Project, Dublin City University 2026
